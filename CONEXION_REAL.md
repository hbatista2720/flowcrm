## 🚀 Setup Completo

### 1. Instalar Dependencias REALES
```bash
npm install whatsapp-web.js puppeteer qrcode-terminal
```

### 2. Instalar ngrok
```bash
brew install ngrok
ngrok config add-authtoken TU_TOKEN
```

### 3. Ejecutar con Internet
```bash
# Terminal 1: App
npm run dev

# Terminal 2: Exponer
ngrok http 3000
```
